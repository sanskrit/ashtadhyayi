BALAMANORAMAS

January 1, 0001

जातुयदोर्लिङ्।जातु यत् अनयोः प्रयोगे अनवक्लृप्त्यमर्,योर्लिङ् स्यादित्यर्थः। यदिति विभक्तिप्रतिरूपकमव्ययम्। अनवक्लृप्त्यमर्षयोरकिंवृत्तेऽपीति लिङ्?लृटौ प्राप्तौ लिङेवेत्यर्तमिदम्। तदाह— लृटोऽपवाद इति। यदायद्योरिति। यदायद्योः प्रयोगेऽपि उक्तविषये लिङ उपसङ्ख्यानमित्यर्थः। जात्वादिशब्दा उदाहरणे अनवक्लृप्त्यमर्षद्योतकाः। त्वादृशो हिरं निन्देदित्यतेतन्नावकल्पयामि, न मर्षयामि वेत्यन्वयः। नावकल्पयामीत्यस्य न संभवायामीत्यर्थः। लृङ् प्राग्वदिति। भविष्यति नित्यं लृङ्भूते वेत्युक्तमिहाप्यनुसंधेयमित्यर्थः, जात्वादियोगस्तु, अनवक्लृप्त्यमर्षयोश्च लिङ्?निमित्तत्वादिति भावः।
