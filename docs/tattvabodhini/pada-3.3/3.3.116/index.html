January 1, 0001

कर्मणि च। कर्तुरिति कर्मणि षष्ठी, उभयप्राप्तौ कर्मणि इति नियमात्। येनेति तृतीया। तदाह– येन स्पश्यमानस्य कर्तुरिति। ल्युट् स्यादिति। नपुंसके भावे इति बोध्यम्। अत एवाह– पूर्वेणेति। ल्युट् चेत्यनेनेत्यर्थः। नेहेति। स्नानकर्तृत्वेऽपि स्नापने न गुरुः कर्ता, किंतु शिष्य इत्यर्थः। शरीरग्रहणं किम् ?। पुत्रस्य परिष्वजनं सुखम्। मानसी प्रीतिरत्र।
