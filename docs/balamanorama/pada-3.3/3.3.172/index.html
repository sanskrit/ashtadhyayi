BALAMANORAMA

3.3.172 शकि लिङ् च

शकि लिङ् च। शक् इति भावे क्विबन्तम्। शक्तौ गम्यमानायामित्यर्थः। भारं त्वं वहेरिति। वोढुं शक्त इत्यर्थः। माङि लुङिति व्याख्यातमपि क्रमप्राप्तं विशेषविवक्षया पुनः स्मार्यते। ननु सर्वलकारापवादोऽयमित्युक्तम्, एवं सति मा भवतु मा भविष्यतीति कथमिति शङ्क्यते – कथमिति। परिहरति– नायं माङिति। किंत्विति। ङकारानुबन्धविधिर्मुक्तस्यापि अव्ययेषु पाठादिति भावः। नच माशब्दमादाय मा भवत्वित्यादिप्रयोगसत्त्वे माङि लुङिति व्यर्थमिति वाच्यं, सर्वलकारविषये न माङ्योगे इत्यडाड्रहितलुङन्तप्रयोगार्थं तदावश्यकत्वात्।
