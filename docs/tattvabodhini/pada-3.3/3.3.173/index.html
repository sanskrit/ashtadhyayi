TATTVABODHINI

3.3.173 आशिषि लिङ्लोटौ

ननु विध्यादिसूत्र एव संप्रश्नप्रार्थनाशीःष्वित्युच्यतां किमनेन पृथक्सूत्रकरणेनेति चेत्। अत्राहुः— इह हि क्तिच्क्तौ चेत्युत्तरसूत्रार्तमाशिषीति तावदावश्यकम्। लिङ्लोटावपीहैव विधेयौ, विध्यादिसूत्रे आशीग्र्रहणे हि स्मे लोडित्यादिना स्वविषये परत्वादद्बाधः स्यात्। सिद्धान्ते तु परत्वादेष विधिः स्मे लोरडित्यादेर्बाधक इति महान्विशेष इति॥
