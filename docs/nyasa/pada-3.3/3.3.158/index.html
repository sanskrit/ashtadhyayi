NYASA

3.3.158 समानकर्त्तृकेषु तमुन्।

पूर्वेण लिङ्लोटोः प्राप्तयोस्तुमुन् विधीयते। तुमुनप्रकृत्यपेक्षमेवइति। यस्माद् धातोस्तुमुन् , अतः प्रत्यासत्तेस्तदपेक्षमेव समानकर्त्तृकत्वं ज्ञायते– तुमुन्प्रकृतेर्यः कत्र्ता स चेदिच्छार्थानामपि भवतीति। इच्छति भोक्तुम् इति। योऽत्रेषेः कत्र्ता स एव भुजेरपीत्यस्तीह समानकर्त्तृकेषु लिङलोटौ च परत्वात् प्राप्नुतस्तौ मा भूतमित्येवमर्थम्, {अर्थः-मुद्रितः पाठः,} ण्वुल् मा भूदित्येव {अर्थश्च-मुद्रितः पाठः} च। देवदत्तं भुञ्जानमिच्छति इति। अत्र भूजेर्देवदत्तः कत्र्ता, इच्छतेस्ततोऽन्य इति नास्ति समानकर्त्तृकत्वम्, तेन वत्र्तमानविवक्षायां लडेव भवति। अनभिधानात् इति। न ह्रत्र तुमुनोत्पद्यमानेन विवक्षतोऽर्थः शक्यतेऽभिधातुमिति। अबिधानलक्षणाश्च कृत्तद्धितसमासाः। अथ वा– अनभिधानात् इति। इच्छन् कर्त्तुमित्यभिधानस्याभावादित्यर्थः। लोकप्रसिद्धानां हि शास्त्रेऽन्वाख्यानं क्रियते, न त्विच्छन् कर्त्तुमित्यादयो लोकप्रसिद्धाः॥
