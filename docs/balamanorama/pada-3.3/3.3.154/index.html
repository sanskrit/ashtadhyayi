BALAMANORAMA

3.3.154 संभावनेऽलमितिचेत्सिद्धाऽप्रयोगे

संभावने। अलमर्थोऽत्र प्रौढिरिति। पर्याप्तिरित्यर्थः। विपरिणम्यते इति। संभावने इति यत् सप्तम्यन्तं तदावर्त्त्य प्रथमया संभावनमिति च विपरिणम्यते, यत्तु अलमिति प्रथमान्तं तदावर्त्त्य सप्तम्या अलमि इति च विपरिणम्यत इत्यर्थः। छान्दसं विभक्तिव्यत्ययमाश्रित्येति शेषः। तथा च संभावने इति सप्तम्यन्तं, संभावनमिति प्रथमान्तं च लभ्यते। तथा अलमिति प्रथमान्तमलभि इति सप्तम्यन्तं च लभ्यते। तत्र संभावने इति सप्तम्यन्तमर्थनिर्देशपरं। तदाह– संभावनेऽर्थ लिङ्?स्यादिति। उत्कटान्यतरकोटिकं ज्ञान संभावनमित्युच्यते। संभावनमितिप्रथमान्तं तु अलमिति प्रथमान्तेन विशेष्यते। इति र्हेतौ। तदाह– तच्चेदिति। तत् = संभावनम्?, अलं = पर्याप्तिहेतुकं चेदित्यन्वयः। षिधधातोर्ज्ञानार्थकात् मतिबुद्धीत वर्तमाने कर्मणि क्ते सिद्धशब्दः। सिद्धे गम्यमानेऽप्यलमर्थे अप्रयोगो यस्य स सिद्धाऽप्रयोगस्तस्मिन्निति विग्रहः। अलमीति सप्तम्यन्तमत्र विशेष्यसमर्पकं संबध्यते। अलंशब्दप्रयोगं विनापि तदर्थे गम्यमाने इति यावत्। तदाह– सिद्धाऽप्रयोगे सतीति। अलमीति शेषः। अपि गिरिमिति। बलवन्तं पुरुषमधिकृत्य अत्युक्तिरियम्। प्रायेण शिरसा गिरि भ#एत्तुमयं समर्थ इत्यर्थः। गिरिभेदसंभावनस्य सामर्थ्यहेतुकत्वद्योतकोऽपिशब्दः। अत्र लिङ्?निमित्तसत्त्वात् क्रियातिपत्तौ भूते लृङ्। अलमिति संभावने सिद्धाऽप्रयोगश्चेदिति सुवचम्।
