BALAMANORAMAS

January 1, 0001

क्षिप्रवचने लृट्। वचनग्रहणात्क्षिप्रपर्याये इति लभ्यते। तदाह– क्षिप्रपर्याये इति। पूर्वविषये इति। आशंसायामित्यर्थः। आशंसायां भूतवच्चेत्यस्यापवादः। ननु क्षिप्रवचनेनेत्येतावतैव आशंसायां क्षिप्रपर्याये उपपदे भविष्यति न भूतवन्न वर्तमानवदिति लाभाल्लृङ्गहणमनर्थकमित्यत आह– नेति वक्तव्ये इति। क्षिप्रवचने नेत्युक्ते `सामान्यातिदेशे विशेषानतिदेशः इनि न्यायेन भविष्यत्सामान्ये विहितस्य लृट एव निषेधः स्यान्नतु लुटः, तस्यानद्यतनभविष्यद्विशेषविधानात्। शीघ्रं वप्स्याम इति। अनद्यतनत्वद्योताय �आश्शब्दः। अत्र न लुडिति भावः।
