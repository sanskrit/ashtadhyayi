3.2.112 अभिज्ञावचने लृट्

अभिज्ञा। भूते इत्यधिक्रियते। अनद्यतने लडित्यतोऽनद्यतन इति वर्तते। अभिज्ञा - स्मृतिः, सा उच्यते बोध्यतेऽनेनेति विग्रहः। तदाह–स्मृतिबोधिनीत्यादि। वत्स्याम इति। अवसामेत्यर्थः। पश्य मृगो धावतीत्यत्र वाक्यार्थरूपमृगकर्तृकधावनमिव स्मरसीत्यत्र गोकुलाधिकरणकाऽस्मत्कर्तृकभूतानद्यतनवासो वाक्यार्थः कर्म। कृष्णेति संबोधनं स्मरणक्रियायां विशेषणम्। तथाच पूर्वोक्तनिवासकर्कं कृष्णसंबोध्यकं स्मरणमिति फलितोऽर्थः।
