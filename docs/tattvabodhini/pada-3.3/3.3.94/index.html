TATTVABODHINI

3.3.94 स्त्रियां क्तिन्

स्त्रियां क्तिन्। घञोऽपवाद इति। यत्तु प्रक्रियाग्रन्थं व्याचक्षाणा आहुः–भाष्ये त्वधिकः कारोऽधिकार इति घञप्युदाह्मत’ इति, तद्रभसात्। न हि तत्र स्त्रीत्वविशिष्टो भावोऽर्थः, किंतु पुंस्त्वविशिष्ट इति ध्येयम्। अजपौ त्विति। तयोरवकाशः–चयः। लवः। क्तिनोऽकाशः–कृतिः। ह्मतिः। चिनोतिस्तौतिभ्यां तु स्त्रियामुभयप्रसङ्गे परत्वात् क्तिन्नित्यर्थः। [प्रामादिकम्। स्फायः स्फी निष्ठायाटमिति निष्ठायामेव स्फीभावविधानादिति भावः]। श्रुयजीति। परत्वात्करणाधिकरणयोश्चेति ल्युटि प्राप्ते वचनम्। श्रुतिः -श्रोत्रम्। यजेरिति। यजेः क्तिनिवचिस्वपीति संप्रसारणेव्रश्चभ्रस्जेति षत्वे इष्टिरिति रूपम्। इज्यतेऽनयेति विग्रहःओ। एवमिष्यतेऽनया इष्टिः। स्तूयतेऽनया स्तुतिः। तेन नत्वमिति। कीर्णिरित्यादौरदाभ्यामित्यनेन नत्वम्। लूनिरित्यादौ तुल्वादिभ्यःइत्यनेनेति विवेकः। ह्लाद इति।ह्लादो निष्ठायामित्यत्रेत्यर्थः। चूर्तिरिति।ति चेति चरफलोरुत्वेर्वोरुपधाया दीर्घः` इति दीर्घः। चायतेः। चायृ पूजानिशामनयोः]। क्तिन्नपीति। अस्त्रियामिति प्रतिषेधेन स्त्रियां वासरूपविधेरबाधादिदमुक्तम्।
