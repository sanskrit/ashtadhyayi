3.3.126 ईषद्दुःसुषुकृच्छ्राऽकृच्छ्रार्थेषु खल्

ईषद्दुःसुषु। इह सामान्योक्तावपि योग्यताबलाद्विशेषणस्य विषयविभागो लभ्यते। दुरिति कृच्छ्रार्थे,इतरौ त्वकृच्छ्रार्थे।तदेतद्दर्शयति–कृच्छ्रे दुष्कर इत्यादिना। भवतेति। न लोकेति षष्ठीनिषेधात्कर्तरि तृतीया। कच्छ्रेत्यादि किम् ?। ईषत्कार्यम्। स्वल्पं कार्यमित्यर्थः। खलिति। लकारः स्वरार्थः, खित्करणं तूत्तरत्र मुमर्थम्। निमिमीति। मीनातिमिनोतिदीङां ल्यपि च विभाषा लीयतेः इति प्राप्तं निषिध्यते। निमय इत्यादि।एरच्। उपसर्गात्खल्घञोः। लभेश्चेत्यनेनैव सिद्धे नियमार्थमिदमित्याह–उपसर्गादेवति।
