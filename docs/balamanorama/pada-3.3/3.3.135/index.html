3.3.135 भविष्यति मर्यादावचनेऽवरस्मिन्

भविष्यति मर्यादा। अवरस्मिन्निति च्छेदः। अनद्यतनवन्नेति। लुण्नेत्यर्थः। अक्रियाप्रबन्धार्थमिति भाष्यम्। असामीप्यार्थं चेति कैयटः। योऽयमिति। क�स्मश्चिज्जनपदविशेषे वसतः आपाटलिपुत्राद्योऽयं गन्तव्योऽध्वा तस्य अध्वनो मध्यवर्तिन्याः कौशाम्ब्याः यदवरं पूर्वप्रदेशस्तत्र सक्तून् �आः प्रभृति पास्याम इति योजना। अत्र कौशाम्ब्या इति मर्यादा गम्यते। अवरमित्यनेन अवरत्वं गम्यते। अत्र भविष्यत्यद्यतेन लुण्न, किंतु लृडेवेति भावः। कालविभागे। पूर्वसूत्रमिति। भविष्यति मर्यादावचनेऽवरस्मिन्निति सूत्रमित्यर्थः। ननु कालमर्यादायामपि पूर्वसूत्रेणैव सिद्धमित्यताअह– अहोरात्रेति। तथा च आहोरात्रसंबन्धिनि प्रविभागे भविष्यति मर्यादावचने इत्युक्तविधिर्न भवतीत्यर्थः। ननुभविष्यति मर्यादावचनेऽवरस्मिन्कालविभागे चानहोरात्राणामित्येकमेव सूत्रं कुतो नेत्यत आह– योगविभाग उत्तरार्थ इति। इदमुत्तरसूत्रे स्पष्टीभविष्यति। योऽयं वत्सर आगामीति। कालतो मर्यादायामुदाहरणम्। आग्रहायण्या इति। मार्गशीर्षपौर्णमास्या इत्यर्थः। युक्ता इति। नियमयुक्ता इत्यर्थः। अध्येष्यामहे इति। अत्र न लुट्, किंतु लृडेवेति भावः। पञ्चदशरात्रयुक्ता इति। पञ्चदश रात्रयो यस्य पक्षस्येति बहुव्रीहिः। अच्प्रत्यन्ववपूर्वादित्यत्र अजिति योगविभागादच्समासान्तः। यद्वापञ्चदशानां रात्रीणां समाहारः पञ्चदशरात्रः।अहःसर्वैकदेशेत्यच्समासान्तः।सङ्ख्यापूर्वंरात्रं क्लीबमिति तुलिङ्गमशिष्यं लोकाश्रयत्वाल्लिङ्गस्ये`ति वचनान्न भवति।
