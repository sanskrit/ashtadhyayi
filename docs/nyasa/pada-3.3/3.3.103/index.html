NYASA

3.3.103 गुरोश्च हलः।

धातोः (3.1.91) इत्यधिकाराद्गुरुग्रहणं हल्ग्रहणं च तस्यैव विशेषणं विज्ञायत इत्याह--गुरुमान् हलन्तो यो धातुःइत्यादि। कथं पुनर्गुरोरित्युच्यमाने गुरुमान् धातुर्लभ्यते? सामथ्र्यात्। न हि हलन्तस्य धातोः कस्यचिद्गुरुसंज्ञा विहिता, उच्यते चेदं गुरोरिति, तत्र सामथ्र्याद्गुरुमान् धातुर्गुरुसंज्ञाकेनाभेदोपचारं कृत्वा गुरुशब्देनोक्त इति गम्यते। इष्यते ह्रभेदोपचाराद्विनापि मतुपा तद्वतोरभिधाम्-- यथा-- शुक्लः पट इति। अथ रुधादिभ्य एव यथा स्यात्, कुडिप्रभृतीभ्यो मा भूदित्येवमर्थो विपर्ययः कस्मान्न विज्ञायतेगुर्वन्तो यो धातुर्हल्वान्इति? ज्ञापकात्, यदयंगत्यर्थकर्मणि द्वितीयाचतुथ्र्यौ चेष्टायामनध्वनि(2.3.12) इत्यप्रत्ययान्तस्य चेष्टाशब्दस्यशकि शङ्गायाम्(धा।पा।८६) इति च शङ्काशब्दस्य निर्देशं करोति, ततो ज्ञायते-- नात्र विपर्यय इति। विपर्यये हि न हि चेष्टेति गुर्वन्तः, नापि शङ्केति।कुण्डाइत्यादि।कुडि दाहे(धा।पा।२७०),हुडि सङ्घाते(धा।पा।२६९),ईह चेष्टायाम्(धा।पा।६३२)ऊह वितर्के` (धा।पा।६४८)– इत्येषां रूपाणि॥
