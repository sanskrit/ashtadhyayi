3.2.78 सुप्यजातौ णिनिस्ताच्छील्ये

उष्णभोजीति। उष्णं भोक्तुं शीलमस्य। आमन्त्रयितेति। मत्रि गुप्तपरिभाषणे, चुरादिराङ्पूर्वः। इदित्त्वान्नुम्, ताच्छील्यस्य विवक्षितत्वातृ?न्। अतएव न लोकेति निषेधात् ब्राआहृणानित्यत्र कर्मणि षष्ठी न कृता। उपसर्गभिन्न एवेति। सत्सूद्विषेति सूत्रे उपसर्गेऽपीत्युक्तत्वादुपसर्गभिन्नस्यैव सुपो लाभायाऽस्मिन्सूत्रे पुनः सुब्ग्रहणं कृतमिति भावः। भाष्यविरोधादिति। उक्तं च भाष्ये– सुबिति वर्तमाने पुनः सुब्गर्हणं किमर्थमनुपसर्ग इत्यव तदभूत्, इदं तु सुब्मात्रे यथा स्यादुदासारिण्यः प्रत्यासारिण्य इति। अस्यायमाशयः— सत्सूद्विषे ति सूत्रे सुपि स्थः इत्यतः सुपीत्यनुवर्तते, तच्चौपसर्गेतरपरम्, उपसर्गेऽपीति पृथगुक्तेः। तदिहानुवर्तमानमर्थाधिकारादुपसर्गेतरपरमेव स्यादिति। निष्कर्षे तु मा भूदिह सुब्वचनग्रहणम्, उपसर्गेऽपीत्यं शस्याप्यनुवृत्त्या निर्वाहात्। सर्वथापि सुब्मात्रे उपपदे णिनिर्नत्वनुपसर्ग एवेति सिद्धान्तः। एतच्च प्रायुङ्क्त– लिङ्गशेषविधिव्र्यापी विशेषैर्यद्यबाधितः’ इ [ ती] ति दिक्।
