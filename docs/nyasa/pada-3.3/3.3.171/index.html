NYASA

3.3.171 कृत्याश्च।

किमर्थम्ित्यादि चोद्यम्।विशेषविहितेनइति परीहारः।कत्र्तरिइत्यादिना परीहारविघटनञ्च।केचित्ित्यादिना परकीयमतेन परोहारसमर्थनम्। भव्यगेयम्िति।समुदायेषु हि वृत्ताः शब्दा अवयवेष्वपि वत्र्तन्ते(शा।प।८५) इति भव्यगेयादयः शब्दा अवयवभूतेषु कृत्येष्विह वत्र्तन्ते।कर्त्तृवाचिनः`इति। भव्यगेयादिसूत्रेणैषां पक्षे कत्र्तरि निपातनात्। योगविभाग उत्तरार्थः॥
