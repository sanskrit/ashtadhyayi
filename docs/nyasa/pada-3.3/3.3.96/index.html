NYASA

3.3.96 मन्त्रे वृषेषपचमनविदभूवीरा उदात्तः।

वृषु मृषु सेचने (धा।पा।७०६,७०७), {इष--धा।पा।} इषु इच्छायाम्} (धा।पा।१३५१), डुपचष् पाके (धा।पा।९९६), मन ज्ञाने (धा।पा।११७६), विद ज्ञाने (धा।पा।१०६४),भू सत्तायाम् (धा।पा।१), वी गतिव्याप्तिप्रजनादिषु (धा।पा।१०४८), रा {दाने -धा।पा।} आदाने (धा।पा।१०५७)। प्रकृतिप्रत्ययोः इति। प्रकृतिसामान्यविवक्षायां द्विवचनम्, अन्यथा हि बहुत्वात् प्रकृतीनां बहुवचनं स्यात्। विभक्तिविपरिणामेन सम्बन्धः इति। वृषादिभ्यो या प्रथमा तस्या अर्थात् पञ्चमीपरिणामः। तेन प्रकृतिप्रत्ययोः सम्बन्धः; अन्यथा सम्बन्धानुपपत्तेः। मतिः इति। अनुदात्तोपदेश (6.4.37) इत्यादिनाऽनुनासिकलोपः। सर्वत्र इत्यादि। मन्त्रे च, अमन्त्रे च – सर्वत्र वृषादिभ्यः किन्विहितः। पचेरपि स्थागादि (3.3.95) सूत्रेण। यद्येवम् किमर्थमिदमित्याह– उदात्तार्थम् इति। इषेस्तु इत्यादि। इषेरिच्छेति निपातनं वक्ष्यति; तस्मात् क्तिन्विधीयते, अन्यथा निपातनेनैवास्य स्यात्। वृषादिभ्यः क्तिन्प्रत्यय एव विधातव्यः। उदात्तवचनमुत्तरार्थम्। व्रजयजोर्भावे क्यप् (3.3.98) इति क्यबुदात्तो यथा स्यात्॥
